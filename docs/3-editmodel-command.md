# Edit model command

- Aliases : **em** **edit**

## Arguments

### model
- description : Name of the model
- type : string

### action
- description : Name of the action , 'remove' or 'add'
- type : string

### columnName
- description : Name of the column to affect
- type : string

## Basic Usage

To edit a model type

```sh
    editModel <model> <action> [columnName]
```
