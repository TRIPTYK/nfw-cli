# GenerateRoute
Generate a basic route.
## Usage:
```sh
$ nfw generate-route <prefix> [methods..]
```
## Alias(es):
genroute
## Options
### --all
- Description: Generate a route for each of all methods.
- Type: boolean
- Default: false
- Example:
```sh
$ nfw generate-route <prefix> [methods..] --all 
```
